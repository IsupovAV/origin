Для сборки проекта под windows, с использованием CMake необходимо:
1. В командной строке создать папку build в каталоге task5 (mkdir build)
2. Перейти в него из папки task1 (cd build)
3. Выполнить команду cmake ..
4. Выполнить команду cmake --build .
5. В каталоге \task3\bin\Debug лежит исполняемый файл task5.exe и 
	 geometry.dll, в каталоге \task5\lib\Debug статическая библиотека geometry.lib.
