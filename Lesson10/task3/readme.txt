Для сборки проекта под windows, с использованием CMake необходимо:
1. В командной строке создать папку build в каталоге task1 (mkdir build)
2. Перейти в него из папки task1 (cd build)
3. Выполнить команду cmake ..
4. Выполнить команду cmake --build .
5. В каталоге \task3\bin\Debug лежит исполняемый файл client_dll.exe и 
	 leaver.dll, в каталоге \task3\lib\Debug статическая библиотека leaver.lib.
