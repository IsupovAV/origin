Для сборки проекта под windows необходимо:
1. В командной строке создать папку build в каталоге task1 (mkdir build)
2. Перейти в него из папки task1 (cd build)
3. Выполнить команду cmake ..
4. Выполнить команду cmake --build .
5. В каталоге \task1\bin\Debug лежит исполняемый файл task1.exe, в каталоге
   \task1\lib\Debug статическая библиотека.
6. Чтобы работать с кирилицей переключить командный интерпретатор chcp 1251 (Кирилица в windows)
