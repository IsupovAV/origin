Проект собирался под windows в среде visual studio 2022
Для сборки проекта под windows необходимо, открыть файл
task4.sln в среде visual studio и пересобрать решение
В каталоге x64/debug лежит запускаемый клиент dll (task4.exe
и сама библиотека geometry.dll и статическая библиотека подключаемая
к проекту task4 geometry.lib)
