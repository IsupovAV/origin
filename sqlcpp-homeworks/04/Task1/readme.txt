Файлы:
create_base.sql - создание самой базы music_site и ее таблиц
insert_data.sql - заполнение, созданных таблиц на предыдущем шаге
create_full.sql - объединенный скрипт, создаёт и заполняет базу
music_site.png - схема взаимосвязей таблиц music_site

Запуск любого скрипта, осуществляется из SQL Shell (psql) postgres
по команде: \i [путь к файлу]/[нужный скрипт].sql
Лучше использовать \i [путь к файлу]/create_full.sql
